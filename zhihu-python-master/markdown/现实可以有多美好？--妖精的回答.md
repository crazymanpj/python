# 现实可以有多美好？
### 作者: 妖精  赞同: 5323
忍不住要答一下，此刻我躺在神经内科病房的躺椅上，旁边是千辛万苦终于被我哄睡着的亲爹。而我的心情是愉悦的。  
感觉我爹就是一出苦逼电视剧的男一号，三年前查出肺癌，我一顿痛哭，然后异地手术。手术过程中，医生一身血乎呲啦的跑出来，一脸傲娇的跟我说，手术很成功，怕你们家属
担心，先出来说一下，还要等缝合和醒麻醉，病人要等好久才能出来。当时觉得阳光照进了了10楼没有窗子的手术等待区。医生满身反射着温暖的阳光，如沐春风，不过如此了
。倒霉的是，术后愈合不好,再来二次手术，上次的左上肺袖式手术追加成了左肺全切。术后一个月开始6期化疗，我爹一路争气的挺下来了 一直到今年8月24号深夜，也就
是前几天，我还在跟朋友聊天说，感觉这两年日子过得很幸福，夫妻恩爱，双方父母健康，儿子调皮可爱。然而，生活的操蛋之处在于，总是在你感恩的时候甩你两个大耳刮子。
25号一早，我还乐滋滋的在干手头的工作，邻居哥哥打电话告诉我，我爹突发脑梗，住院了，情况不好。我拨通我妈的电话，我妈语无伦次。我拨通老公的电话，这次换我语无
伦次。还好，老公清醒，安排我赶紧打车回家收拾行李，他也从单位赶回来，我俩直接回我家。（我是独生女外嫁，苦乐参半啊）四个小时的路程，我老公开了3个小时到家，我
庆幸他还保持理智，一路劝着我，在安全的前提下，尽量快，我们不能出事，安全是大前提，我感谢他的冷静果断。  
到医院，我爸连病房都没有，拖了关系，在走廊上加个小床。我爸就那么躺着，目光涣散，神志不清。然后，医生找我谈话，左脑大面积脑梗塞，要我们做好两手准备。医院会尽
力救治，但是由于堵塞是面积较大，没有完全的把握。那一刻，我头脑是空白的，只有老爸那双涣散的眼睛在漂浮。几天治疗下来，解除了生命危险的警报，看着老爸原来住院时
处于瘫痪状态的左手可以靠着肩关节的力量摇动，左腿可以抬起来一会。我觉得，这操蛋的生活还是有奔头的。  
今天，老爸由于躺了很多天，一直闹脾气。吃饭时间，我问：稀饭吃吗?他摇摇手，问：面条吃吗？他摇摇手……所有的答案都是摇手，最后，我忍无可忍，问他：羊肉串吃吗？
我亲爹他老人家炫酷的歪嘴一笑，点着头比了一个1。说，就吃1串，不要辣椒。我忍不住，哈哈哈一通狂笑。瞬间，对我爹的康复又充满信心。晚上10点，我跟我爹在一屋子
都是糖尿病高血压忌口忌的看到活鸡都两眼放绿光的病人里，吃烤串，谈人生。这个是妥妥的拉仇恨啊。（我爹一直血压低，医生没要求忌口，为了哄老头高兴，给他吃了一串羊
肉串）  
此刻的我，觉得生活还是温柔的对待着我，不管怎样，此刻，他活着，我伴在左右。生活可以有多美好，不过如此。（9月2日夜）++++++++++++++++++++

这两天老爸精神好了一些，我觉得这次生病对我爸智力影响比较大，然后我亲爹开启了神作模式。躺着久了不舒服，于是半小时一次的坐起，躺下，左翻，右翻，轮流无限循环。
把我跟老妈折腾的双双腰疼，蹲起坐下都要直着腰板，我跟老妈打趣说，老爸让咱俩练深蹲呢，深蹲好，深蹲翘臀。我妈被我说的揉腰的红花油洒了一地。  
老爸是个军事政治迷。没生病前就一直念叨着要看阅兵，看阅兵，看阅兵，重要的事情说三遍。可是，从住院一直忙乱，就忘了这茬了。早上十点，才伺候好我爹吃饭，他老人家
忽然说，我要看阅兵，今天3号了。医院的无线网超慢，但是勉强能连接。他老人家就凑着我的手机看，由于很久没关注，我也不知道是直播还是重播，看到习大大接待外国领导
，我爸激动的直咳嗽，然后各个方队步伐整齐，唰唰的走着正步，我歪着脑袋边看边感慨，我擦，好整齐，我擦，兵哥哥太帅了，我正独自陶醉着呢，我爸忽然激动异常，呜呜的
哭了，久久无法平复。劝了好久，好容易平复下来，习大大一句：同志们好，同志们辛苦了。兵哥哥们整齐的回答：为人民服务！然后我爹又激动的哭了。  
我无从解读，我爹的眼泪表达了什么，是觉得自己生病错过了阅兵，还是被阅兵的壮观场面所震撼，他只能像个孩子一样呜呜的哭泣来表达内心的澎湃，还是想起了他当年在青海
当兵的峥嵘岁月。（我爹一直很骄傲，他在青海当了多年汽车兵，心情好的时候总喜欢吹牛他在青海的光辉岁月。）  
先写到这吧，老头这会睡着了，我要抓紧时间休息会。漫漫长夜，我们家老头不知道还要怎么整我呢，各位午安。（9月3日，午）

猜猜，老头这会在干嘛?  
哈哈，躺在床上，用着懒人手机架斗地主！我大家自行想象一下吧，不想暴露老爷子如此猥琐形象的照片。眼看着几局下去，以前挣的积分赔进去不少。  
老爸脾气坏，又开始不好好吃饭，早上查房血压低到98。我有点着急上火，然后恐吓我爸说，要是不好好吃饭，就喊医生给你插胃管，到时候不论吃不吃，直接打进去。然后，
我爸气得翻身给我个后背，任凭怎么喊都不理我了。  
中午，老蜜们说慰劳奋战在抗击脑梗一线的妇女斗士，中午给我加个餐。然后出去吃了顿好的，现在吃多了想睡觉，就拿手机忽悠我爸。  
另外，从昨4天开始让老爸尝试站起来了，感觉他左腿左手还是用不上力气，全靠我跟我妈一左一右的撑着才能站住。反正，也不着急了，慢慢来吧。  
刚刚我爸同病房一大叔今天办出院，收拾完东西临走了，跟病房里的病友告别，到我爸床头，大叔语重心长的对我爹说：老弟啊，好好治疗吧，你看你闺女，来的时候咋看都是一
城市知识分子，现在都累成老妇女了。尼玛啊，我还没来及得意一下，就一颗心碎成渣了。老妇女，呃，呃，呃……  
（9月5日，午。）

汇报一下近况吧，我爹目前转至康复科仍旧在积极训练，目前已经从轮椅过渡到了拐杖，还在练习走路的阶段。我闺蜜把家里一套原来用于出租的一楼房子，借给我爹妈住了，国
庆节把家搬好了。我以前的家在5楼，我爹估计是爬不上去。现在的单位在我请假期间，老板干了好多恶心事。总之是希望我自动离职。我也打算不行就辞职拉倒，两地跑跑照顾
我爹，到明年再重新找工作。  
总之，一切没那么糟糕。生活，还是TMD挺美好。（10月12日，午。）
