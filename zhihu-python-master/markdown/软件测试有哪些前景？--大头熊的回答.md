# 软件测试有哪些前景？
### 作者: 大头熊  赞同: 2
软件测试的风云人物属于编程高手，顶级如开发出selenium, jenkins这些从根本上改变软件测试方式和效率的工具；次级的是开发公司内部工具和系统，能够
把开源工具整合到公司的开发测试流程中的人，如谷歌，脸书都有自己专门的开发测试平台，也会不时开源内部开发的工具。  
  
在软件行业混，不会编程很难有大发展，当然也有马云这样的异数，不过人家的路数和绝大部分人不一样，不在我们这里讨论之列。谷歌等一流的技术公司连产品和项目经理都要
求要计算机科班出身。
