# 珠海这座城市怎么样？
### 作者: 包子  赞同: 45
赞一楼的答案！我是跟着爸妈从北方迁移到珠海的，在这所城市生活了差不多十年，然后又再次离开。离开珠海以后，才知道自己被珠海“富养”了那么久。以至于把很多事情当
做理所当然。  
  
一直以为城市里满眼的绿色是理所应当的，蓝天白云是应该天天能看见的，以为大海就在家门口很正常，以为沙滩都是免费的。每次坐大巴到广州，一下车看见灰不溜秋的天和摩
肩接踵的人，马上想转身掉头走时，就特别想珠海。还有去北京的时候从下飞机开始就连着咳嗽了5天。  
  
想出行的话到广州深圳都很方便，而且当天来回没问题。去澳门就是抬抬脚走过去的事儿。去香港坐船也就1个半小时。所以嫌弃珠海没地儿逛街的话随时随地都可以走，晚上还
能扛着大包小包回来按个脚吃个宵夜再碎觉。现在有了城际轻轨更方便鸟。  
  
在珠海长大也是一件很幸福的事情呐。很多小孩子都是像我一样跟父母一起移民过来的，所以用普通话沟通没问题，不会被人排斥，又能多学一门粤语。这里也很重视英语教学，
九几年的时候我上小学一年级就要学英语了，还很喜欢看tvb明珠台的外国动画片和930电影。珠海的师资也很棒，因为很多老师都是从内地过来的，尤其是湖北。另外珠海
是12年义务教育。上学那会儿最喜欢刮台风，因为可以放假。还有中学生校服是水手服和中山装&gt;_&lt;.简直是福利好吗。  
  
生活在珠海简直会各种磨灭人的斗志啊...
